Features

Two small components. One big quality-of-life fix.

Windows ships no SVG codec, so Explorer and Photos are blind to vector files. winSVG fills the gap — natively, quietly, and without touching anything it shouldn't.

🖼️

Explorer thumbnails

A COM shell extension rasterizes .svg and .svgz with the Skia engine, so folders show real, transparency-aware previews at every icon size.

⚡

Double-click viewer

A lightweight viewer with Chromium-quality rendering via WebView2 — what you see is exactly what a browser renders.

🔍

Zoom & pan

Mouse-wheel zoom around the cursor, drag to pan, double-click toggles fit ↔ 100%. Vector-sharp at any magnification.

📤

Export to image

Export PNG / JPG / WebP at ×1, ×2, or ×4 scale — with a white, black, or transparent background.

✂️

Crop tool

Crop overlay with aspect presets — Free, 1:1, 16:9, 21:9, 9:16, 4:3, 3:4 — then export the cropped region razor-sharp at any scale.

📁

Folder navigation

Arrow keys flip through every SVG in the folder, with file name, pixel dimensions, and position shown in the toolbar.

🎨

Background toggle

Cycle white → checkerboard → black to inspect transparency and light-on-dark artwork properly.

🔒

No admin, clean uninstall

Installs per-user with zero admin rights. Uninstall from Settings → Apps restores your previous file association and removes every registry entry.

🆓

Free & open-source

MIT licensed. Read the code, build it yourself, or contribute on GitHub.

Get started

Installed in under a minute

Requires Windows 10 / 11 (x64) and the free .NET 8 Desktop Runtime (the installer checks and points you to it if missing).

Download & run the installer

Grab winSVG-Setup.exe below. No admin prompt — it installs only for your user account.

Open a folder of SVGs

Thumbnails just work. Press F5 if the folder was already open.

Make it your default viewer (optional)

Right-click any .svg → Open with → Choose another app → winSVG → Always. Windows requires this single manual confirmation — no installer can legitimately skip it.

Stop squinting at generic file icons

Download the latest release — free, MIT licensed, uninstalls cleanly.
